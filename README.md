# 🃏 ホールデム 役判定クイズ

テキサスホールデムの勝敗を瞬時に判断する力を鍛えるWebクイズアプリです。

**▶ [アプリを開く](https://anzai-0916.github.io/poker-quiz/)**

---

## 概要

2人のプレイヤーの手札とコミュニティカードが表示され、どちらが勝つかを10秒以内に回答します。間違えた問題は自動で復習リストに保存されます。

## 機能

- ランダムな手札・ボードの自動生成
- 7枚から最強5枚を選ぶ役判定ロジック（C(7,5)=21通りを全探索）
- 10秒カウントダウンタイマー（ON/OFF切り替え可）
- 連続正解数（ストリーク）のトラッキング
- 間違えた問題の復習リスト（LocalStorageで永続化）
- 勝敗の解説文を自動生成
- Xへのスコアシェア機能（Web Share API対応）

## 使用技術

| 技術 | 用途 |
|------|------|
| HTML / CSS / JavaScript | フロントエンド（`index.html` / `style.css` / `script.js` の3ファイル構成） |
| LocalStorage | 復習データの永続化 |
| Web Share API | スマホからのXシェア |
| GitHub Pages | ホスティング |

## ローカルでの実行

```bash
git clone https://github.com/anzai-0916/poker-quiz.git
cd poker-quiz
# index.html をブラウザで開くだけで動作します
```
